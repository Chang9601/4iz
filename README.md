# 4iz

## 소개

- 다국적 스포츠 의류 기업 나이키를 모델링한 4iz

## 기술 스택

- 개발
  - 프레임워크: Nest.js
  - 데이터베이스: MySQL(TypeORM)
  - 테스트: Jest(Unit, E2E)
- 인프라
  - 컨테이너화: Docker
  - 클라우드: AWS(EC2, RDS, VPC(Private & Public Subnet, IGW, NACL))
  - CI: GitHub Actions
  - 로드 밸런싱: NGINX(HTTPS)
- 툴
  - GitHub
  - Postman
